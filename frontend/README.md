# CapstoneBook Frontend

Next.js App Router frontend. Source được chia theo feature tại `src/features`; `src/app` chỉ chịu trách nhiệm routing và metadata.

```powershell
npm run dev
npm run lint
npm run build
```

Biến môi trường công khai duy nhất hiện dùng là `NEXT_PUBLIC_API_URL` trong `.env.local`.
