# CapstoneBook Backend

NestJS API cho authentication, catalog MongoDB, PayOS, thư viện và tiến độ đọc.

```powershell
npm run start:dev
npm run lint
npm run build
npm test
```

Module nghiệp vụ nằm trong `src/modules`. Secret được đọc từ `backend/.env.local` rồi `.env` ở root; không đặt secret trong source.
